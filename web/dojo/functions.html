Reuse the same code

Below you will see two separate buttons. Despite what they say, they actually are not all that different. Both buttons actually call the same exact function when they are clicked.

Chain of functionality

Functions, of course, can be called from within other functions. Often times, the called function is called a helper function. Helper functions are designed to be small and usually only accomplish a single task.

You'll also notice that some of the functions even have stuff being passed into them. That is data that the function can use.
